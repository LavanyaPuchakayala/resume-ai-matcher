# resume-ai-matcher
Built an AI-powered job matching web app using React and Vite. Users can upload resumes, and the app extracts skills to match them with relevant jobs showing a % match score. Deployed live on Netlify.
md# AI Resume Job Matcher

An AI-powered job portal that helps candidates match their resumes to relevant job listings.

**Key Features:**
- Resume Upload & Parsing: Extracts skills from PDF/DOC resumes
- Smart Job Matching: Shows jobs with % Match score based on skills
- Search Filters: Search by job title, skills, location
- Responsive UI: Works on mobile and desktop
- Dark/Light Mode: Toggle for better UX
- Fast Deployment: Hosted on Netlify

**Tech Stack:** React, Vite, Tailwind CSS, JavaScript
