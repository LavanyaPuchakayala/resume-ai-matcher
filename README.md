# AI Resume Job Matcher

AI-powered job portal that matches resumes to jobs.

## Features
- Upload PDF/DOC resume and auto-extract skills
- Smart job matching with % Match score  
- Dark/Light mode
- Deployed on Netlify

## Live Demo
https://resume-ai-matcher.netlify.app
